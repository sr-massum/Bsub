# BSub - বাংলা সাবটাইটেল টুলকিট

## 🌟 বৈশিষ্ট্যসমূহ

# BSub Bundle

A complete toolkit for downloading, translating, and synchronizing Bengali subtitles.

## Features

### Subtitle Downloader

Download subtitles directly from Subsource using the integrated API.

🌐 Website: https://bsub.rf.gd

### Subtitle Translator

Translate subtitle files into Bangla and other supported languages.

🌐 Website: https://bsub-translator.rf.gd

### Subtitle Synchronizer

Adjust subtitle timing and synchronize subtitle files with videos.

🌐 Website: https://subtitle-sync.rf.gd



## Telegram Bot

Search and download subtitles directly from Telegram.

🤖 Bot Username: **@BsubAssist_Bot**

Features:

* Search subtitles instantly
* Download subtitles directly from Telegram
* Fast and user-friendly interface
* Supports Bangla subtitle discovery

Open Telegram and start the bot:

https://t.me/BsubAssist_Bot
